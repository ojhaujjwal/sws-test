## Description
SWS API Test.

## Installation

It's recommended to use nvm to setup the same node version. If not, any node on v14.* should work.

```bash
$ nvm use

$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
